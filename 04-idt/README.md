# IDT and ISRs

## Interrupt Descriptor Table 

A.K.A. IDT

Interrupts may come from :

- hardware: keyboard, serial port, timer,
- CPU send interrupts when program when wrong
- softwar interrupts

The IDT describes a handler for each interrupt.
